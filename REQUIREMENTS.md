# Frame Maker — 摄影相框编辑器 需求文档

## 1. 产品概述

我个人使用的 Web App，用于为自己的摄影作品添加带有拍摄参数的相框。  
上传照片后，可选择相框模板、编辑模板内的文字内容，实时预览效果，最终导出带相框的成品图。

---

## 3. 核心概念：相框模板

每个相框模板是一个**独立的、自描述的配置对象**，包含：

- **底色**（白 / 黑 / 其他）
- **文字区域定义**：每个模板自行决定显示哪些文字字段、字段的位置和样式
- **文字内容**：模板自带默认占位文字，用户可编辑

## 4. 功能需求

### 4.1 图片上传

- 支持点击上传或拖拽上传照片
- 支持 JPG / PNG / WebP 格式
- 上传后自动读取图片尺寸，自适应预览区
- 未上传时显示占位引导界面（带上传图标和提示文字）

### 4.2 实时预览区

- 位于页面**上方主体区域**
- 实时渲染：照片 + 相框 + 文字的合成效果
- 预览画面等比缩放，适配屏幕
- 使用 Canvas 绘制，确保所见即所得（预览与导出一致）

### 4.3 相框选择器

- 位于预览区**右侧面板**内
- **纵向列表**排列，无分类标签
- 当前占位：白底模板 + 黑底模板 两个选项
- 每个选项显示为缩略卡片（带缩略预览效果）
- 选中态有明确的视觉高亮（边框/阴影）
- 切换时预览区实时更新

### 4.4 文字编辑区

- 位于预览区**下方**
- 编辑区的字段**由当前选中的模板决定**（不同模板可有不同字段）
- 当前两个占位模板的字段均为：

| 字段 | 默认占位值 |
| ---- | --------- |
| 焦距 | 23mm      |
| 光圈 | f/1.8     |
| 快门 | 1/1000    |
| ISO  | 100       |

- 修改后预览区实时更新

### 4.5 导出功能

- 点击「导出」按钮，将 Canvas 渲染结果导出为图片文件
- 导出格式：PNG（默认）
- 导出分辨率与原图一致（非预览缩放尺寸）
- 自动触发浏览器下载

## 5. 页面布局（Web 端）

采用**左右分栏 + 底部编辑**布局，预览为核心，右侧选模板，底部编辑文字：

```
┌──────────────────────────────────────────────────────────────┐
│  Frame Maker                          [上传照片]  [导出]      │  ← 顶部工具栏
├────────────────────────────────────────────┬─────────────────┤
│                                            │                 │
│                                            │  相框选择器      │
│              ┌───────────────────────┐     │                 │
│              │                       │     │  ┌───────────┐  │
│              │       照片预览         │     │  │  白底模板   │  │
│              │                       │     │  │  (缩略图)  │  │
│              ├───────────────────────┤     │  └───────────┘  │
│              │  23mm f/1.8 1/1000 …  │     │  ┌───────────┐  │
│              └───────────────────────┘     │  │  黑底模板   │  │
│                                            │  │  (缩略图)  │  │
│                                            │  └───────────┘  │
│                                            │                 │
├────────────────────────────────────────────┴─────────────────┤
│                                                              │
│  焦距 [23mm   ]   光圈 [f/1.8  ]   快门 [1/1000 ]   ISO [100] │  ← 文字编辑区
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

- **左侧主体（约 75% 宽度）**：灰色背景预览区，居中显示带相框的照片卡片
- **右侧面板（约 25% 宽度）**：相框模板纵向列表，缩略卡片可滚动
- **底部编辑区**：横跨全宽，根据当前模板动态显示对应文字字段
- **顶部工具栏**：应用名称 + 上传/导出按钮

## 6. 视觉规范

### 6.1 主题

- **亮色主题**
- 背景色：浅灰（预览区）+ 白色（工具栏、编辑区）
- 强调色：待定（建议沉稳的深色系，用于选中态和按钮）

### 6.2 字体

- 界面 UI：系统默认无衬线字体
- 相框水印文字：无衬线字体

### 6.3 圆角与阴影

- 相框卡片：圆角 8px，轻微投影
- 选择器缩略卡片：圆角 6px
- 按钮：圆角 6px

### 6.4 间距

- 基础单位 4px（4/8/12/16/24/32/48）

## 7. 交互细节

| 场景             | 交互行为                                    |
| ---------------- | ------------------------------------------- |
| 首次进入         | 预览区显示上传引导（图标 + 提示文字）         |
| 上传图片         | 图片加载后自动应用默认相框（白底），预览即时渲染 |
| 切换相框         | 预览区实时切换到新模板样式                    |
| 编辑文字         | 内容变化时实时更新预览                        |
| 导出             | 按钮点击后自动下载 PNG                       |
| 无图片时点击导出  | 提示「请先上传照片」                         |

## 8. 模板扩展机制

每个模板为一个配置对象，结构示意：

```typescript
interface FrameTemplate {
  id: string
  name: string
  // 相框底色
  backgroundColor: string
  // 信息栏高度占比（相对于图片高度）
  barHeightRatio: number
  // 文字字段定义
  fields: {
    key: string
    label: string
    defaultValue: string
  }[]
  // 渲染函数：负责在 Canvas 上绘制该模板的信息栏
  render: (ctx: CanvasRenderingContext2D, area: Rect, values: Record<string, string>) => void
}
```

新增模板只需添加新的配置对象，无需修改其他代码。
